# Gold Rate Billing App 💰

A Flask-based web application for gold shops to manage daily rates, customer registrations, and billing processes.

### ✨ Features
* **User Authentication:** Secure Login, Registration, and Password Reset functionality.
* **Dashboard:** View and manage gold/silver rates.
* **Billing Management:** Add, Edit, and View billing records.
* **Responsive UI:** Clean interface built with HTML5 and CSS3.

### 📂 Project Structure
* `app.py`: Main Flask application logic.
* `templates/`: Contains all HTML files (Login, Register, Dashboard, etc.).
* `static/`: Contains CSS, JavaScript, and Images.

### 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** (Mention your database like SQLite/MySQL)

### 🚀 How to Run
1. Install Flask: `pip install flask`
2. Run the app: `python app.py`
3. Open `http://127.0.0.1:5000` in your browser.
